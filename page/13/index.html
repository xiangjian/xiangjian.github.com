I18n,l10n的含义

Dec 2nd, 2006 | Comments

经常在英文网页中看到i18n和l10n这样的词语，不知道是什么含义，一直也没有留意。今天看到python的tools文件夹中有一个叫i18n的文件夹，就想知道这个到底是什么意思。

结果Google了一下

define:i18n

I18N:即internationalization，软件国际化。一个过程，通过修改代码实现代码的文化独立性，即代码不依赖于任何特定文化的信息(cultural specific information)。

L10N:即localization，软件本地化。一个过程，为每一种支持的人类语言来定制软件。

其中　18、10是表示单词中的省略的字母数。

让文字在同行内显示

Oct 25th, 2006 | Comments

以前我遇到几个对象在同一行内显示的时候，通常是将它们放置在一个Table的中。
感觉很是不爽，table是用来显示表格数据的，不是用来布局的—-一个CSS的前辈说的。
仔细看了一下CSS的文档，发现有一个属性可以实现。
display版本：CSS1/CSS2　 兼容性：IE4+ NS4+　继承性：有
语法：
display : block | none | inline | compact | marker | inline-table | list-item | run-in | table | table-caption | table-cell | table-column | table-column-group | table-footer-group | table-header-group | table-row | table-row-group
取值：
block :　 CSS1 块对象的默认值。将对象强制作为块对象呈递，为对象之后添加新行
none :　 CSS1 隐藏对象。与 visibility 属性的hidden值不同，其不为被隐藏的对象保留其物理空间
inline :　 CSS1 内联对象的默认值。将对象强制作为内联对象呈递，从对象中删除行
inline-block :　 IE5.5 将对象呈递为内联对象，但是对象的内容作为块对象呈递。旁边的内联对象会被呈递在同一行内
compact :　 CSS2 未支持。分配对象为块对象或基于内容之上的内联对象
marker :　 CSS2 未支持。指定内容在容器对象之前或之后。要使用此参数，对象必须和 :after 及 :before 伪元素一起使用
inline-table :　 CSS2 未支持。将表格显示为无前后换行的内联对象或内联容器
list-item :　 CSS2 将块对象指定为列表项目。并可以添加可选项目标志
run-in :　 CSS2 未支持。分配对象为块对象或基于内容之上的内联对象
table :　 CSS2 未支持。将对象作为块元素级的表格显示
table-caption :　 CSS2 未支持。将对象作为表格标题显示
table-cell :　 CSS2 未支持。将对象作为表格单元格显示
table-column :　 CSS2 未支持。将对象作为表格列显示
table-column-group :　 CSS2 未支持。将对象作为表格列组显示
table-header-group :　 CSS2 将对象作为表格标题组显示
table-footer-group :　 CSS2 将对象作为表格脚注组显示
table-row :　 CSS2 未支持。将对象作为表格行显示
table-row-group :　 CSS2 未支持。将对象作为表格行组显示
使用dispaly:inline;可以让对象在同行内显示，但是不能使用Float属性了，因为Float是针对block对象的。

使用sc 命令的一个怪问题？

Oct 18th, 2006 | Comments

今天我想把svn添加为系统服务，按照说明写了如下的命令：
sc create svn binpath=”"D:\program\svn\bin\svnserve.exe" –service -r "D:\data\svndemo"” displayname=”Subversion Repository” depend=Tcpip start=demand
结果不能成功。
试了好几种方法还是不行，到微软的网站上看它的例子，仔细对比发现，它的每个”＝”后都有一个空格。于是将每一个的”＝”后面加一个空格就可以执行了。
sc create svn binpath= “"D:\program\svn\bin\svnserve.exe" –service -r "D:\data\svndemo"” displayname= “Subversion Repository” depend= Tcpip start= demand

ASP中使用类

Oct 16th, 2006 | Comments

在写ASP页面时，有时候需要将一系列实现特定功能，而又能在比较多的页面中使用的时候，这时常常，需要将这些功能封装起来。通常的方法是写COM组件，这个方法开发较慢，而且我也不会，使用ASP的VBscript的Class来实现的。

Class lang
Private objDoc
private path
‘private langtext
Private Sub Class_Initialize
path=Server.MapPath(“/”)&”\lang.xml”
set objDoc=Server.CreateObject(“Microsoft.FreeThreadedXMLDOM”)
objDoc.async=false
objDoc.load path
End Sub
Public Function lang(id)
set langnode=objDoc.SelectSingleNode(“Cnodclang/Lang[@id=’”&id &”’]”)
if langnode is nothing then
langre=”obj is not exist”
else
if(Session(“language”)=”en”) then
langre=langnode.getAttribute(“En”)
elseif(Session(“language”)=”cn”) then
langre=langnode.getAttribute(“Zh”)
else
langre=langnode.getAttribute(“Zh”)
end if
Response.Write langre
end if
End Function
Public Property Get langtext(id)
set langnode=objDoc.SelectSingleNode(“Cnodclang/Lang[@id=’”&id &”’]”)
if langnode is nothing then
langre=”obj is not exist”
else
if(Session(“language”)=”en”) then
langre=langnode.getAttribute(“En”)
elseif(Session(“language”)=”cn”) then
langre=langnode.getAttribute(“Zh”)
else
langre=langnode.getAttribute(“Zh”)
end if
langtext=langre
end if
End Property
Public Sub edit_node(id,lang,value)
set langnode=objDoc.SelectSingleNode(“Cnodclang/Lang[@id=’”&id &”’]”)
if langnode is nothing then
set root=objDoc.SelectSingleNode(“Cnodclang”)
if root is nothing then
else
set new_node=objDoc.createElement(“Lang”)
new_node.setAttribute “id”,id
new node.setAttribute lang,value
root.appendChild(new_node)
end if
else
langnode.setAttribute lang,value
end if
objDoc.save path
End Sub
Public Sub del_node(id)
set langnode=objDoc.SelectSingleNode(“Cnodclang/Lang[@id=’”&id &”’]”)
if langnode is nothing then
else
set root=objDoc.SelectSingleNode(“Cnodclang”)
if root is nothing then
else
root.removeChild(langnode)
end if
end if
End Sub
Private Sub Class_Terminate()
set objDoc=nothing
End Sub
End Class
总结下：
1.ASP使用Class_Initialize，Class_Terminate()类似构造函数和析构函数。
2.ASP类函数不能使用return ,因此函数没有返回值（作用和SUB一样）。
3.对成员的函数的赋值是通过let 来实现的。
4.如果要返回值，则使用Property GET 来完成的。
