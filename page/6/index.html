花18分钟计划好你的一天

Aug 18th, 2009 | Comments

工作的时候，如何让自己每天的八个小时工作发挥最大的作用。说实话，每天的八小时一直集中精神真的不容易，像我，一天有很多时间都不在做与工作相关的事情（遗憾啊）。下面是Peter Bregman写的，如何管理好一天的八小时，提高工作效率。每天抽出十八分钟来管理计划每一天，不要让时间不知道怎么就过去了。
原文http://blogs.harvardbusiness.org/bregman/2009/07/an-18minute-plan-for-managing.html
这十八分钟是怎么分配的呢：
1. ５分钟做好一天的计划。
2. 每一个小时花１分钟巩固计划。
3. 最后花５分钟总结一天工作成果。

    常见的恶习，我占了几个

    Aug 17th, 2009 | Comments

    原文：http://www.psytopic.com/mag/post/how-to-overcome-bad-habits.html
    以下是Marshall(全球高级领导者教练领域的先驱与权威)总结的在工作生活中常见20个恶习，具体来说，这些行为包括：
    　　● 求胜欲太强：在任何情况下都要不惜一切代价去打败对方，无论这样做是否值得。
    　　● 太喜欢增加价值: 每次讨论的时候总要发表一番自己的高见。
    　　● 太喜欢点评：总是要对别人的说法评论一番，把自己的标准强加于人。
    　　● 总是发表破坏性评论：总是为了让自己的言语听起来更深刻、更诙谐而说出一些不必要的讥讽之语。
    　　● 喜欢用”不”“但是”或”可是”来开头：过多地使用否定式过渡语，实际上是在告诉对方，”你错了，我才是对的”。
    　　● 告诉世界你有多聪明：总是要告诉周围的人我比他们想象的更聪明。
    　　● 乱发脾气：经常在愤怒的时候跟人沟通，把情绪上的波动不加约束地传导出去。
    　　● 负面思维，比如”让我来告诉你这样做为什么不行”：总是要用自己的负面思维去影响周围的人–即使是在毫无必要的时候。
    　　● 隐瞒信息：为了让自己占有一定的心理优势而拒绝跟别人分享信息。
    　　● 不懂得表示认可：不懂得表扬或奖励别人。
    　　● 喜欢抢功：总是过于高估自己在某项工作中的作用。这是最让人厌恶的一种恶习。
    　　● 总是喜欢找借口：总是把自己的坏习惯归结为某种无法改变的原因，以此来为自己的行为开脱。
    　　● 把一切都归咎于过去：总是把自己的失误或缺点归咎于以往的人或事。其实是在为自己的过失寻找替罪羊。
    　　● 偏袒下属：不能公正地对待自己周围的人。
    　　● 拒绝道歉：拒绝为自己的行为承担责任，拒绝承认错误，不承认自己的行为影响到了其他人。
    　　● 不懂得聆听：对同事最不礼貌的一种表现。
    　　● 不懂得感激：最缺乏礼貌的一种坏习惯。
    　　● 惩罚报信人：错误地攻击那些本来想要帮助你的人。
    　　● 乱找替罪羊：总是把自己的过失推到其他人头上。
    　　● 过于强调自我：把自己身上那些无法改正的缺点看成是一种美德，总是强调”我就是这样一个人”。

    Postgis在windows平台下的安装

    Aug 10th, 2009 | Comments

    postgis是postgresql的一个GIS扩展，类似于oracle spatial,支持ogc标准，也是目前开源社区中使用最多的GIS空间数据。postgresql是一个强大的开源数据，类似于mysql，比 mysql功能更强大，提供了说许多商业数据库才支持的数据特性，介绍在这PostgreSQL。postgresql在８以后开始对windows提供了很好的支持.
    下面我们在windows平台下安装PostgreSQL:
    1)下载PostgreSQL的安装包，到EnterpriseDB主页下载（EnterpriseDB是一家在PostgreSQL基础上开发的数据库厂商，它们提供的安装文件集成了常用的程序，更好的数据库安装工具）

    2)然后是安装，一路Next,设置一下postgre用户的密码，安装路径，默认使用端口(5432),就可以了。安装完后这个版本自带了pgAdmin III(数据库管理工具)，Stack Builder(数据库插件，应用程序安装工具)。
    ３)我们通过Stack Builder来安装postgis.安装过程也是一路确定，没有什么困难。安装完后，重启数据库，就可以使用pgAdminIII创建空间数据库了。

    Clearing Float

    Aug 10th, 2009 | Comments

    前几天做一个信息显示的界面，布局是左右两边，要这左右两在不同的浏览器情况下都可以自动适应布局，使用div布局方式一直没有很好解决，没有办法只能使用table来做布局。
    后来看到了这篇文章，使用div布局是可以解决这个问题的。
    最终的解决方法是 

    div.container {
border: 1px solid #000000;
overflow: hidden;
width: 100%;
}
div.left {
width: 50%;
float: left;
}
div.right {
width: 50%;
float: right;
}

    五个最好用的系统还原工具

    Aug 9th, 2009 | Comments

    当计算机出现灾难时，备份数据是降低空难损失的最好方法。你是使用什么方法恢复正在使用的的系统呢。下面有５个常用的选择（免费软件）。

    1. Macrium Reflect Free (Windows, Free)
      Macrium Reflect Free是一款不需要发钱，但可以使用Macrium Reflect disk imaging功能的工具。它可以备份你的磁盘数据到网络上的位置，光盘，和其它磁盘上。内置的任务和DVD读写程序可以让你维护任务变得更轻松， Macrium Reflect Free支持微软的影卷拷备服务，实时备份，多版本备份支持。Macrium Reflect Free的另一个亮点是程序完全兼容bartPE可以，以及支持基于linux的启动盘。

    2. DriveImageXML (Windows, Free)

      DriveImageXML创建彻底的磁盘映像，从一个磁盘到另一个磁盘，提供了像原始文件系统一样的浏览备份文件。这就意味着你如果需要从原始备份中还 原一个小文件，不需要还原整个硬盘，只需要抽取这个小文件就可以了。和Macrium Reflect Free差不多，支持微软的影卷拷备服务。

    3. Paragon Drive Backup Free (Windows, Free)

      Paragon Drive Backup是Paragon的商业硬盘软件中的免费的个人版，但是它支持很多种磁盘类型和文件系统。Paragon支持微软影卷服务和实时备份，同时当磁盘坏掉的时候，它也能恢复多个分区和磁盘快照。可以通过闪存或CD/DVD来恢复备份。还有用户界面，如上面的截图，完全是向导式的，对磁盘不熟悉的用户可以轻松上手。

    4. ImageX (Windows, Free)

      ImageX 是被电脑厂商或大型组织用来制作和分发系统到大量的电脑上。它主要是一个命令行工具，因为它在geek中流行，创建了一个GUI系统—-GImageX， 如上面的截图。尽管有不少关键的限制，它只能运行在现代的windows文件系统之上，如果你使用大量的windows系统， ImageX还是有不少优点。映像文件小你得感谢”单实例”模式，如果一个文件出现在不同的位置，映像文件中只会存储一次。ImageX允许重写磁盘，允 许你从映像中还原部分内容。

    5. Clonezilla (Windows/Mac/Linux, Free)

      Clonezilla是一个强大的开源磁盘映像工具，对新用户使用足够简单。研究它深入的特性，尽管会让用户的需要扩展。它的服务器版本支持网络大磁盘映像，它支持大量的文件系统，包括windows,linux,Mac-based machines.如果你使用一个不常用的文件系统，你依然可以使用Clonezilla进行扇区到扇区的备份，用来代替有高效可以压缩文件的支持文件系统的备份。个人版本是一个live CD,可以定制了各种应用，在Clonezilla网站有详细的说明文档。
