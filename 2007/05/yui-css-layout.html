使用YUI CSS创建跨浏览器页面

May 30th, 2007 | Comments

现在在编写WEB应用的时候更多的是要考虑跨浏览器时的的支持，一般情况下使用工具完成的页面一般不是很好，YUI提供了一个很好CSS解决方案。

  1. YUI CSS GRID ：方便的进行页面布局。
  2. YUI CSS FONT :解决不同浏览器之间字体的差异。
  3. YUI CSS RESET：这个是去掉浏览器默认的标签的属性。
