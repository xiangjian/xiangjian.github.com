使用代码语法高亮工具高亮你网站或博客代码中的代码，让你贴出的代码显得更专业可读性更高。下面介绍五个受欢迎的网站博客语法高亮工具。

  1. SyntaxHighlighter
SyntaxHighlighter是一个深受程序员喜爱的在线展示代码片段的工具，展示结果非常漂亮（如下图）。它是基于javascript,无需在服务器端做任何设置。我的blog也是使用这个来做语法高亮的,如果要在blogger上使用这个可以参考我的文章:高亮显示blogger中的代码。
  • Quick Highlighter Quick Highlighter提供一个在线工具，将你的代码生成高亮的html代码，你可以将这些代码粘贴到博客或网站中去。下图是生成效果。
  • FV Code Highlighter
    Frank Verhoeven开发的this great code higlither用以将纯文本代码转化为类似dreamweaver高亮的样式。下面是高亮结果。
  • Source Code Highlighter
    一个在线工具,这个工具与上一个差不多，不过它生成的html不包含CSS，只使用了html标签。
  • Highlight.js
    • Highlight.js 是一个语法高亮工具，它是自动工作的，找到代码片段，识别语言类型自动高亮。 原文：http://woork.blogspot.com/2009/07/interesting-code-highlighters-for-blogs.html
