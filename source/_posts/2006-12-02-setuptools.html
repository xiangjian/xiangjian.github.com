setuptools:它是 Python Enterprise Application Kit（PEAK）的一个副项目。 setuptools 替换了标准的 distutils 库，并为 Python 添加了版本化的包和依赖性管理。Perl 用户比较熟悉 CPAN，而 Ruby 用户则比较熟悉 Gems；引导 setuptools 的 ez_setup 工具和随之而生的扩展后的 easy_install 与 "Cheeseshop"（Python Package Index，也称为 "PyPI"）一起工作来实现相同的功能。另外，setuptools 还让我们可以将自己的库包装成一个称为 "egg" 的包，它与 Java? 的 JAR 文件类似，不过是用于 Python 的。

使用非常简单。
